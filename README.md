# Bean Journal

Bean Journal is a small Next.js app for tracking coffee tastings and storing brew recipes: a personal, lovingly crafted journal for coffee lovers. This README explains the project, how to run it locally, and how to deploy it to Vercel with Supabase + Prisma.

---

## Quick Overview

- Framework: Next.js (App Router)
- Styling: Tailwind CSS + custom global styles
- Fonts: DM Serif Display (headings), Space Grotesk (body), JetBrains Mono (monospace accents)
- Auth: Supabase Auth
- Database: Postgres via Prisma (Prisma client generated to `lib/generated/prisma`)
- Hosting: Vercel
